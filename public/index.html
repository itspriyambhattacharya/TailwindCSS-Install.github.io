How to Install TailwindCSS - A Simple Guide

  1. Setup a New npm Package 
    npm init
  2. Install TailwindCSS 
    npm install -D tailwindcss@latest postcss@latest autoprefixer@latest;
  3. Create Two Folders src and public. Make a file style.css in src folder and Paste this 
    @tailwind base;
@tailwind components;
@tailwind utilities;
  4. Open package.json and Delete This --> "test": "echo \"Error: no test specified\" && exit 1" 
    {
  "name": "folder",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC"
}
  5. In the place of "test": "echo \"Error: no test specified\" && exit 1" place This
    Note: Do not Change Anything Else 
    "build-css": "tailwindcss build src/style.css -o public/css/style.css"
  6. The Final package.json File Should Look Like This 
    {
  "name": "project-2",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "build-css": "tailwindcss build src/style.css -o public/css/style.css"
  },
  "author": "",
  "license": "ISC",
  "devDependencies": {
  "autoprefixer": "^10.2.6",
  "postcss": "^8.3.5",
  "tailwindcss": "^2.2.4"
  }
}
  7. Compile the style.css File
    Note: Run This Command Everytime You Change src/style.css 
    npm run build-css
  8. Create your configuration file - If youâ€™d like to customize your Tailwind installation, generate a config file 
    npx tailwindcss init
